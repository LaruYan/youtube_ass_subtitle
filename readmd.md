# youtube ass subtitle
유튜브에서 .ass 확장자의 자막을 보이게 해줍니다.

# 사용법
1. 확장 프로그램 설치후 유튜브에 들어갑니다.
2. 크롬 확장프로그램에서 youtube ass subtitle을 선택합니다.
3. 팝업창에서 '파일선택' 버튼을 눌러 ass 자막을 선택합니다.
4. load subtitle 버튼을 누르면 자막이 나옵니다.
